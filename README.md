This Java program is a simple input-output system using a basic scanner. 
It allows users to register, login and then add, view, update, and delete records. 
All data is stored temporarily in an ArrayList while the program is running.
Feedback messages help users know when actions are successful or if something is wrong.
The project uses Object-Oriented Programming (OOP) with separate classes for methods.
It is designed for beginners to practice Java and basic data management.
This project is ideal for learning how to combine OOP concepts in Java.
